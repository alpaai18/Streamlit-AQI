## Run steamlit app
- Clone/Download as Zip
- Open PowerShell
- Copy Path Directory Folder zip
- ``` cd 'Path_Lengkap_Folder_Zip' ```
- ``` python -m venv venv ```
- ``` venv\Scripts\Activate```
- ``` pip install -r requirements.txt```
- ``` cd dashboard```
- ``` streamlit run dashboard.py ```
